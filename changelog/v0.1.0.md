# v0.1.0

## Features

- Shortcut `o` opens selected article link in system browser
- Shortcut `g` jumps to top of feed list
- Shortcut `C` collapses all categories in sidebar
- Pre-fetch on hover — silently fetch article content when navigating ArticleList (without waiting for Enter)
- Article age display — article detail shows time since publication in color
- Scrollbar — visual scroll indicator in all scrollable panels

## UI

- Feed names truncate with `…` in sidebar when too long (non-selected)
- Ghost feed/category origin dims during move
- Editor panels use `├─`/`╰─` tree connectors matching the sidebar
- Feed title scroll animation starts faster (3-tick delay instead of 8)
- ArticleDetail keeps the three-panel layout instead of expanding to full width
- Star icon replaced — circle color encodes saved state: yellow=saved, blue=unread, dim=read
- Article fetch spinner appears in detail title during readability fetches
- ArticleList shows MAUVE border when focused; ArticleDetail shows SURFACE0 border in preview mode
- Scrollbar renders in both preview and full-detail modes
- Sidebar, article list, and article detail reserve 1 col for scrollbar so text does not overlap
- SavedCategoryEditor moved out of Settings; now accessible via `e` from the Saved tab sidebar

## Fixes

- Feeds panel shows category headers as colored non-interactive labels
- New-category input no longer double-highlights the category below it
- Category move places item as sibling after cursor (not as child); `←`/`→` adjusts depth during move
- Feed move cursor can land on category headers to place feed as first child
- Category headers show at correct depth (no phantom indent)
- Category no longer highlights on hover during move (only preview arrow shows)
- Category deletion confirmation is now a popup instead of a status bar message
- Editor panels use persistent ListState so scroll does not stick when navigating up from the bottom
- `tree_connector` correctly shows `╰─` for the last child that has children of its own
- Sidebar scroll position preserved when entering ArticleList and returning
- Settings border highlights with MAUVE when focused
- CategoryPicker popup no longer renders a black background
- Saved tab sidebar no longer renders a black background

## Refactor

- Extracted `tree_connector` helper, deduplicating connector logic across `content.rs` and `editor.rs`
- Introduced `ListScroll` struct bundling `cursor + ListState`; replaced `saved_cat_editor_cursor` and added scrollbar rendering to `SavedCategoryEditor` and `CategoryPicker`
